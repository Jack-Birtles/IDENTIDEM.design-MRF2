# Datasheets

Reference datasheets for the LiDAR LDO errata ([../lidar-stage2-ldo-design.md](../lidar-stage2-ldo-design.md)).

| File | Part | In repo? |
|------|------|----------|
| `TI-TLV755P-SBVS320D.pdf` | TI TLV75533 LDO (U1). TI doc SBVS320D, rev Sept 2024. Freely redistributable. | Yes |
| `DTS6012M-Docdt22-Rev1.2.pdf` | Paioujidian DTS6012M dToF sensor. **Vendor-confidential — not committed** (see `.gitignore`). | No, local only |

The DTS6012M datasheet is marked "Confidential" by the vendor, so it is kept on the
maintainer's machine and excluded from this public repo. Links to it in the errata
docs resolve locally for whoever holds the file; get it from the module vendor
(派欧机电 / Paioujidian) if you need it. Source online: <https://www.ti.com/lit/ds/symlink/tlv755p.pdf> for the TI part.
