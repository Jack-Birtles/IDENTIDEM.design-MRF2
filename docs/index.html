Update Steps

  1. Use Chrome or Edge on desktop.
  2. Connect the MRF2 over USB-C data cable.
  3. Click Connect Camera and Install.
  4. Select the MRF2 serial device from the prompt.
  5. If flashing fails to start, hold BOOT, tap RESET, then retry install.
  6. Wait until flashing completes, then reboot the camera.

Troubleshooting

  • No port appears: try a different USB cable or port.
  • If no port appears, open updater with ?allports=1 and retry.
  • If the installer reports device lost, re-enter BOOT/RESET download mode and retry.
  • Adafruit runtime USB ports (0x239A:0x8xxx) are auto-switched to bootloader first; bypass with ?allowruntime=1.
  • If stuck, use the VS Code guide in the project documentation.
