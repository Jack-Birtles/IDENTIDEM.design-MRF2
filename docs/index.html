Update Steps

  1. Use Chrome or Edge on desktop.
  2. Connect the MRF2 over USB-C data cable.
  3. Click Connect Camera and Install.
  4. Select the ESP32-S3 serial device from the prompt.
  5. Wait until flashing completes, then reboot the camera.

Troubleshooting

  • No port appears: try a different USB cable or port.
  • If stuck, use the VS Code guide in the project documentation.
